##Admin commands: /rounds `command` _parameter_

###/rounds `limit` {number}
    sets new rounds points limit

###/rounds `fto` {mm:ss}
    sets finish timeout

###/rounds `wu` {mm:ss}
    sets warmup duration

###/rounds `newrules` {bool}
    set use new rules
